# webUI_localModel

## 介绍
webUI，用于本地LLM大语言模型chatGLM2-6b-int4，AI人工智能聊天chat。免魔法、无需联网、告别openAI_chatGPT_api_key！

## 安装
### 模型
下载https://huggingface.co/THUDM/chatglm2-6b-int4 到 .\model\

### 环境
python -m venv myenv

### 依赖
pip install -r requirements.txt

## 使用
myenv\Scripts\activate  
python app.py

或双击run.bat
